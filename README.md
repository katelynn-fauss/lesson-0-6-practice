# Weekly Sales Summary

A small Python script that summarizes fictional weekly sales data.
Built as part of Lesson 0.4 of Coding Temple's AI Data Analytics
Foundations course.

## What it does

The script defines a dictionary of fictional sales values for six weeks, then prints the total and average across all weeks.

## How to run

You need Python 3.10 or higher. From the terminal, in this folder:

    python practice_notebook.py

(On Mac and Linux you may need `python3` instead of `python`.)

## Files

- `practice_notbook.py` - the main script
- `practice_notebook.ipynb` - the jupyter script
- `README.md` - this file
