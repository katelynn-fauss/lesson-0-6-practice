# Weekly Sales Summary

A small Python script that summarizes fictional weekly sales summary. I built this for a practice work for coding temples lesson 04. 

## What it does

The script defines a dictionary of fictional sales values for six
weeks, then prints the total and average across all weeks.

## How to run

You need Python 3.10 or higher. From the terminal, in this folder:

    python practice_notebook.py

(On Mac and Linux you may need `python3` instead of `python`.)
